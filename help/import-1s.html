Импорт 1С

Возможности системы управления позволяют импортировать данные из "1С: Управление торговлей" в Ваш магазин. Для этого Вам необходимо настроить синхронизацию данных на стороне 1С,  задать значения для опций на стороне системы управления и запустить импорт.

Основные принципы импорта из 1С в Магазин-2:

  • В качестве ключевого поля для номенклатур и предложений при импорте используется универсальный идентификатор 1С. На основе этого идентификатора работают и режимы импорта: "Добавлять новые, изменять существующие", "Добавлять новые, не изменять сущестующие", "Не добавлять новые, изменять существующие", "Удалять существующие", "Скрывать существующие", "Отображать существующие". Под существующими в этих случаях понимаются те товары, которые ранее уже импортировались из 1С в Магазин.
  • В качестве товаров импортируются номенклатуры. Если номенклатура в 1С имеет предложения - первое предложение будет использоваться в качестве основного товара, а все последующие - в качестве модификаций.
  • В качестве названий товаров - импортируются названия номенклатур и предложений соответственно.
  • Также импортируются виды цен, установленные для номенклатур в 1С.
  • В качестве дополнительных полей импортируются характеристики номенклатур и предложений. Характеристики импортируются как дополнительные поля вида "Текстовая строка" : глобальные, не обязательные для заполнения, не участвующие в фильтрации, отображаемые на сайте, экспортируемые в Yandex.Market (в дальнейшем эти параметры можно отредактировать).
  • Данные о производителях/поставщиках номенклатур и предложений также попадают в данные о товарах.
